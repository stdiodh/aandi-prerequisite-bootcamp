# A&I Prerequisite Bootcamp

이 브랜치는 `00-answer` 참고 구현 비교 브랜치입니다.
Spring Boot 코드를 작성하기 전에 HTTP, JSON, Postman, Git, DB 기본 용어를 손으로 익히는 준비 단계의 기준 자료를 담습니다.

## 이번 시퀀스에서 확인하는 것

- GET과 POST 요청의 차이
- 상태 코드와 응답 body 읽기
- JSON body 값 수정 결과
- Git branch를 만드는 이유
- table, row, column, PK 설명
