# Deprecated answer guide

이 문서는 과거 main 브랜치에서 정답 비교 자료로 사용하던 안내입니다.
현재 수업 운영에서는 main 브랜치에 정답성 자료를 두지 않고, 실습 후 필요한 경우 `00-answer` 브랜치에서 비교합니다.

## 현재 기준

- 학생 시작 브랜치: `00-implementation`
- 비교 브랜치: `00-answer`
- main 브랜치 역할: 레포 안내와 Visual Lab 진입점 제공

## 운영 메모

외부에서 이 문서를 직접 링크했을 가능성이 있어 파일은 유지합니다.
정답 예시, 응답 비교 자료, 용어 기준 자료는 `00-answer` 브랜치에서 확인합니다.
