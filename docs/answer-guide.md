# 선수지식 부트캠프 정답 비교 가이드

이 브랜치는 starter이므로 정답 자료를 그대로 싣지 않습니다.  
완성된 비교 기준은 `00-answer` 브랜치에서 확인합니다.

## 비교할 때 볼 파일

- `answer/http/get-post-response.json`
- `answer/http/create-post-response.json`
- `answer/git/branch-why.txt`
- `answer/db/terms.txt`

## 비교할 질문

- GET 요청의 기대 상태 코드와 응답 body를 읽었는가
- POST 요청의 기대 상태 코드와 새 `id`를 확인했는가
- JSON body에서 key와 value를 구분했는가
- branch를 왜 만드는지 설명할 수 있는가
- table, row, column, PK를 표 예시로 설명할 수 있는가

## 강사용 메모

학생에게 정답 파일을 먼저 보여주기보다, 요청과 Git 명령을 직접 실행한 뒤 비교하게 합니다.
