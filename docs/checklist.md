# 선수지식 부트캠프 체크리스트

## 실습자 체크리스트

- [ ] GET과 POST 차이를 설명할 수 있습니다.
- [ ] 상태 코드 `200`, `201`, `400`, `404`의 뜻을 말할 수 있습니다.
- [ ] JSON body에서 값을 직접 수정할 수 있습니다.
- [ ] `git clone`, `git checkout -b`, `git add`, `git commit`을 따라 할 수 있습니다.
- [ ] branch를 만드는 이유를 한 문장으로 설명할 수 있습니다.
- [ ] table, row, column, PK를 회원 표 예시로 설명할 수 있습니다.

## 리뷰어 / PPT 체크리스트

- [ ] HTTP 메서드 예시가 준비되어 있는가
- [ ] 상태 코드 예시가 준비되어 있는가
- [ ] JSON 요청/응답 예시가 준비되어 있는가
- [ ] Git branch 흐름을 단계로 보여줄 수 있는가
- [ ] table, row, column, PK 예시가 준비되어 있는가
- [ ] 다음 시퀀스와 연결되는 이유를 설명할 수 있는가

## 남은 한계와 다음 시퀀스 연결

이번 시퀀스는 Spring Boot 코드를 작성하지 않습니다.
다음 시퀀스에서는 여기서 확인한 HTTP 요청과 JSON 응답을 Controller, Service, Repository 코드 흐름으로 연결합니다.
