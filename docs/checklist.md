# 선수지식 부트캠프 체크리스트

## 요청/응답 확인

- [ ] GET 요청을 보내고 `200 OK`를 확인했습니다.
- [ ] POST 요청을 보내고 생성 응답 상태를 확인했습니다.
- [ ] JSON body의 key와 value를 구분했습니다.
- [ ] 요청 body 값을 바꾼 뒤 응답 변화를 확인했습니다.

## Git 흐름 확인

- [ ] `git clone`으로 레포를 가져왔습니다.
- [ ] `git checkout -b`로 작업 branch를 만들었습니다.
- [ ] `git add`로 기록할 파일을 골랐습니다.
- [ ] `git commit`으로 작업을 저장했습니다.

## DB 용어 확인

- [ ] `starter/db/members-table-diagram.txt`를 열었습니다.
- [ ] table, row, column, PK를 회원 표 예시로 설명했습니다.

## 마무리 확인

- [ ] `java -version`을 실행했습니다.
- [ ] `git --version`을 실행했습니다.
- [ ] 다음 레포에서 `./gradlew bootRun`과 `./gradlew test`를 실행할 준비가 되었습니다.
