## 운영 메모

- 이번 시퀀스는 학생이 코드를 길게 작성하는 단계가 아닙니다.
- 제공 자료는 학생의 손동작을 빠르게 시작하게 돕는 용도입니다.
- 정답 비교는 `docs/answer-guide.md`와 `answer/` 폴더를 함께 사용하면 충분합니다.

## 학생이 직접 작성하지 않는 범위

아래 항목은 운영자가 미리 제공합니다.

- Postman 컬렉션
- Git 최소 명령어 가이드
- HTTP/JSON 예시 입력값
- DB 기초 그림 자료
- 빠른 정답 비교 자료
